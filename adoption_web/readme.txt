HappyPaws.

Hay muchos refugios de animales y hogares de tránsito a lo largo del mundo. Pero decidí centrarme en
Capital Federal para este proyecto. Queriendo crear una forma de listar dichos refugios y hogares para
la comodidad de las personas tanto al adoptar como al dar en adopción. Saber de dónde vienen los animales
y a dónde van. 


Instalación de proyecto - Backend

¿Cómo ejecutar el proyecto?

    - Instalá Python.
    - Cloná el proyecto con https://github.com/SofiHerdol/python_final_project.git
    - Ahora, creá un entorno virtual con "virtualenv" o con "Pipenv" y activalo.
    - Instalá los requerimientos:
        "pip install -r requirements.txt"
    - Ejecutá las migraciones:
        "python manage.py makemigrations"
        "python manage.py migrate"
    - ¡Ejecutá el proyecto!
        "python manage.py runserver"


Tecnologías y lenguajes utilizados para la creación de HappyPaws:
    - Python
    - Django
    - Bootstrap 
    - Html
    - Css (para la parte final del proyecto)
